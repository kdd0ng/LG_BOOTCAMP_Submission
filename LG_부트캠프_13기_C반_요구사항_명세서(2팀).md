# SW_Bootcamp_13기_C반_2팀_요구사항_명세서

## 프로젝트 개요

**Squeeze_PPT2Json** — PowerPoint 보고서 파일(.pptx)을 자동으로 파싱하여 구조화된 JSON 및 Excel 파일로 변환하는 LLM 기반 파이프라인.

---

| R-02 | 복잡한 표가 포함된 슬라이드를 자동으로 감지하고 표 영역을 플레이스홀더로 마스킹한다 | Step 1-1 실행 후 입력 PPTX 각각에 대응하는 마스킹된 `.pptx` 파일이 `src/data/raw_masked/`에 생성된다 | 00:15 |

## 비고

- 시연 타임스탬프는 `시연영상.mp4` 기준이며, 실행 환경(API 응답 속도)에 따라 ±5초 오차 발생 가능
- R-04, R-05, R-06은 외부 LLM API 호출이 필요하므로 단위 테스트는 사전 실행 결과 파일을 기반으로 검증
- R-10~R-13은 웹 서비스(`src/program_frontend/web`) 기능이며, `docker compose up`으로 기동 후 시연 가능
- R-14는 시연영상 대상이 아닌 CI/CD 요구사항으로, GitHub Actions 실행 로그를 기준으로 검증
- R-15는 결과물 품질 요구사항으로, 결과 JSON/Excel 품질 테스트 로그를 기준으로 검증
